# padflux-brand

Padfluxのブランド資産・note運用・SNS情報を管理するリポジトリです。

## ディレクトリ構成

```
padflux-brand/
├── brand/         # ブランドコンセプト・トーン&マナー
├── note/          # note クリエイタープロフィール・記事原稿
└── social/        # SNSアカウント情報
```

## 概要

- **Padflux**：ブラウザで動く音楽ツールのプラットフォーム
- **現在のプロダクト**：[Padflux Synth](https://padflux.com/synth)
- **運営者**：コードを書けないドラマー × Claude Code
